## ✨ Features

* 🧭 Multiple locations (Town, Store, Cave)
* ⚔️ Combat system with randomness
* 👾 Multiple enemies (Slime, Beast, Dragon)
* 💰 Gold & XP progression
* 🛒 Weapon upgrade system
* 🎒 Inventory management
* 🎲 Hidden Easter egg mini-game
* 🔄 Replay system

---

## 🧠 How the Game Works

* You start in the **town square**
* Earn **gold & XP** by defeating enemies
* Buy **weapons & health** from the store
* Fight stronger monsters in the cave
* Defeat the **dragon** to win 🏆

## 🌐 Deploy on GitHub Pages (IMPORTANT 🚀)

1. Push your project to GitHub
2. Go to:

   * **Settings → Pages**
3. Under "Source":

   * Select **main branch**
   * Folder: `/root`
4. Click **Save**

👉 Your game will be live at:

```
https://your-username.github.io/dragon-repeller/
```
